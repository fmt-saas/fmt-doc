# Edition de textes

## Correcteur orthographique et grammatical 

### Principe de fonctionnement

L’éditeur de texte utilisé dans le module **AG** dispose d’une aide à la correction orthographique.
Cette correction ne dépend pas directement du logiciel lui-même : elle utilise les **fonctionnalités de correction du navigateur internet** (Chrome, Edge, Firefox, etc.).

Selon la configuration du navigateur, celui-ci peut :

- souligner les **fautes d’orthographe**
- parfois détecter certaines **erreurs grammaticales**

Cependant, par défaut, les navigateurs détectent principalement les **fautes d’orthographe** et beaucoup moins les **erreurs de grammaire**.

C’est la raison pour laquelle certaines fautes grammaticales peuvent ne pas être signalées.



### Améliorer la détection des fautes de grammaire

Pour bénéficier d’une correction plus avancée (grammaire, style, reformulation), il est possible d’installer une **extension de correction linguistique dans le navigateur**.

Ces extensions fonctionnent automatiquement dans les zones de texte du logiciel.

### Extensions recommandées

#### LanguageTool

Extension open-source permettant la correction :

- orthographique
- grammaticale
- stylistique

Installation :
https://languagetool.org/fr/chrome

Avantages :

- bonne prise en charge du français
- gratuit dans sa version de base
- respecte davantage la confidentialité des données



#### Grammarly

Extension très connue pour la correction avancée de texte.

Installation :
https://www.grammarly.com/browser/chrome

Remarque :
la correction du français est plus limitée que pour l’anglais.



#### QuillBot

Extension proposant :

- correction grammaticale
- reformulation de texte

Installation :
https://quillbot.com/app/chrome-extension




## Vérifier que la correction du navigateur est activée

Dans certains cas, la correction orthographique du navigateur peut être désactivée.

Dans Chrome :

1. Ouvrir **Paramètres**
2. Aller dans **Langues**
3. Vérifier que **Correction orthographique** est activée
4. Ajouter **Français** dans les langues vérifiées
