# Documentation FMT

Bienvenue dans la documentation fonctionnelle de FMT.

Ce site centralise les informations utiles pour comprendre le produit, ses processus métier et son fonctionnement au quotidien.

## Objectif

Cette documentation a pour but de :

- fournir un point d'entrée clair pour les équipes ;
- décrire les principaux modules et workflows ;
- faciliter l'onboarding et la transmission des connaissances ;
- servir de référence lors des opérations courantes.

## Organisation

Les contenus sont structurés par domaine : copropriété, comptabilité, budget, documents, suivis, utilisateurs, infrastructure et permissions.

Chaque section présente les concepts clés, les étapes opérationnelles et, lorsque nécessaire, les liens vers des pages détaillées.

## Bonnes pratiques de lecture

- Commencer par les pages d'introduction de chaque domaine.
- Utiliser la navigation latérale pour accéder aux sous-thèmes.
- Mettre à jour les pages concernées dès qu'un processus évolue.
